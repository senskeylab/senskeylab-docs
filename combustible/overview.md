# 가연성 가스 센서 개요

가연성 가스 센서(Combustible Gas Sensor)는 메탄(CH₄), 프로판(C₃H₈), 수소(H₂) 등 폭발·연소 위험이 있는 가스의 농도를 측정하는 센서입니다. 가스 누출 감지 및 산업 안전 분야에서 폭발 하한계(LEL, Lower Explosive Limit) 기준 농도를 주요 측정 단위로 사용합니다.

---

## LEL이란

가스마다 공기 중 연소 가능한 농도 범위가 정해져 있으며, 그 하한값이 **LEL(폭발 하한계)**, 상한값이 **UEL(폭발 상한계)**입니다. 산업 현장에서는 LEL의 10~25%를 경보 기준으로 설정하는 것이 일반적입니다.

| 가스 | LEL (%vol) | UEL (%vol) |
|------|-----------|-----------|
| 메탄 (CH₄) | 5.0 | 15.0 |
| 프로판 (C₃H₈) | 2.1 | 9.5 |
| 부탄 (C₄H₁₀) | 1.8 | 8.4 |
| 수소 (H₂) | 4.0 | 75.0 |
| 에탄올 (C₂H₅OH) | 3.3 | 19.0 |
| 암모니아 (NH₃) | 15.0 | 28.0 |
| 아세틸렌 (C₂H₂) | 2.5 | 100.0 |

> **측정 단위:** 가스 검지기는 통상 0 ~ 100% LEL 범위로 표시합니다. 예를 들어 메탄 2.5%vol = 50% LEL로 표시됩니다.

---

## 감지 방식 종류

가연성 가스 감지에는 주로 세 가지 방식이 사용됩니다.

| 방식 | 원리 | 주요 특징 |
|------|------|-----------|
| **접촉연소식** (Catalytic Bead) | 촉매 표면 연소열 → 저항 변화 | 정확도·선형성 우수, 산업 표준 |
| **반도체식** (MOS) | 가스 흡착 → 반도체 저항 변화 | 저가·소형, 선택성 낮음 |
| **적외선식** (NDIR) | 가스 분자의 IR 흡수 | 고정밀, 피독 없음, 고가 |

이 페이지에서는 산업용 고정식 검지기에 가장 많이 쓰이는 **접촉연소식**을 중심으로 설명합니다.

---

## 접촉연소식 동작 원리

접촉연소식 센서는 **휘트스톤 브리지(Wheatstone Bridge)** 회로를 기반으로 동작합니다.

**반응 메커니즘**

가연성 가스가 촉매 코팅된 비드(Detector bead) 표면에 도달하면 저온 산화 연소가 일어납니다.

```
CₙHₘ + O₂ ──[촉매, ~500°C]──→ CO₂ + H₂O + 열(Q)
```

발생한 연소열이 비드 내부의 코일 저항을 상승시키며, 이 저항 변화를 브리지 회로로 검출합니다.

**브리지 회로 구성**

```
        Vcc
         │
    ┌────┴────┐
    │         │
   [D]       [C]       D: Detector bead (촉매 코팅)
    │         │         C: Compensator bead (비촉매, 온도 보상)
   ─┴─       ─┴─
   [R1]     [R2]      R1, R2: 고정 저항
    │         │
    └────┬────┘
         │
        GND

출력 = V(D 측) - V(C 측) → 가스 농도에 비례한 차동 전압
```

가스가 없을 때 D = C이므로 출력은 0 V. 가스 농도가 높아지면 D의 저항이 증가하여 출력 전압이 상승합니다.

> **Compensator bead 역할:** 촉매를 제거한 동일 구조의 비드로, 주변 온도·습도 변화에 의한 D bead의 저항 변동을 상쇄하여 안정적인 측정을 가능하게 합니다.

## 센서 셀 구조

```
  ┌─────────────────────────────────────────┐
  │          가스 유입 (확산 또는 흡인)        │
  └──────────────┬──────────────────────────┘
                 │
  ┌──────────────▼──────────────────────────┐
  │            소결 금속 필터 (방폭망)         │  ← 역화 방지, 이물 차단
  └──────────────┬──────────────────────────┘
                 │
  ┌──────────────▼──────────────────────────┐
  │      ┌────────────┐  ┌────────────┐     │
  │      │ Detector   │  │Compensator │     │  ← 휘트스톤 브리지
  │      │ bead (D)   │  │ bead (C)   │     │
  │      │ [촉매코팅]  │  │ [비촉매]   │     │
  │      └────────────┘  └────────────┘     │
  └─────────────────────────────────────────┘
                 │
  ┌──────────────▼──────────────────────────┐
  │         신호 처리 회로                    │  ← 브리지 구동, 차동 증폭, ADC
  └─────────────────────────────────────────┘
```

| 구성 요소 | 역할 |
|-----------|------|
| **Detector bead** | Pt 코일 위에 Al₂O₃ + 귀금속 촉매(Pd, Pt) 코팅. 가스 연소열 감지 |
| **Compensator bead** | 촉매 미코팅 동일 구조. 온도·습도 드리프트 보상 |
| **소결 금속 필터** | 방폭 구조의 핵심. 센서 내부 연소가 외부로 전파되지 않도록 차단 |
| **브리지 회로** | D, C bead + 고정 저항 2개로 구성. 차동 출력 |

## 주요 가스별 특성

아래 수치는 대표적인 접촉연소식 센서의 사양 범위입니다. 감도는 메탄(CH₄) 기준 교정 후 각 가스의 상대 감도로 환산하는 방식이 일반적입니다.

| 측정 가스 | 측정 범위 | T₉₀ 응답시간 | 상대 감도 (CH₄=1 기준) | 기대 수명 |
|-----------|----------|-------------|----------------------|----------|
| 메탄 (CH₄) | 0 ~ 100% LEL | ≤ 30 s | 1.00 | 3 ~ 5년 |
| 프로판 (C₃H₈) | 0 ~ 100% LEL | ≤ 30 s | ~0.55 | 3 ~ 5년 |
| 부탄 (C₄H₁₀) | 0 ~ 100% LEL | ≤ 30 s | ~0.50 | 3 ~ 5년 |
| 수소 (H₂) | 0 ~ 100% LEL | ≤ 20 s | ~0.45 | 3 ~ 5년 |
| 에탄올 (C₂H₅OH) | 0 ~ 100% LEL | ≤ 30 s | ~0.60 | 3 ~ 5년 |
| 아세틸렌 (C₂H₂) | 0 ~ 100% LEL | ≤ 30 s | ~0.70 | 3 ~ 5년 |

> **상대 감도 환산:** 센서는 통상 CH₄으로 교정(calibration)되며, 다른 가스를 측정할 때는 상대 감도 계수(Response Factor)를 적용하여 실제 LEL%로 환산합니다. 각 가스의 정확한 계수는 제품 데이터시트를 참조하십시오.

## 장단점 및 타 방식과 비교

### 감지 방식별 비교

| 항목 | 접촉연소식 | 반도체식 (MOS) | 적외선식 (NDIR) | 전기화학식 |
|------|-----------|--------------|--------------|-----------|
| **주요 적용** | 가연성 가스 LEL | 가연성·독성 다양 | 가연성 가스, CO₂ | 독성 가스, O₂ |
| **측정 범위** | 0 ~ 100% LEL | ppm ~ % | ppm ~ % | ppm ~ %vol |
| **선형성** | 우수 | 보통 | 우수 | 우수 |
| **선택성** | 보통 (가연성 가스 전반) | 낮음 | 높음 (가스 고유 IR) | 높음 |
| **소비 전력** | 높음 (~500 mW) | 보통 (히터) | 보통 | 매우 낮음 |
| **응답 속도** | 빠름 (10~30 s) | 빠름 | 빠름 | 보통 (30~60 s) |
| **수명** | 3 ~ 5년 | 3 ~ 5년 | 5년 이상 | 1 ~ 3년 |
| **촉매 피독** | 있음 (Si, S, Pb) | 없음 | 없음 | 일부 가스 |
| **가격** | 보통 | 낮음 | 높음 | 보통~높음 |
| **방폭 구조** | 소결 필터 필수 | 케이스 설계 | 광학계 설계 | 소결 필터 필요 |

### 접촉연소식 장점

- **산업 표준:** 수십 년간 가연성 가스 감지의 기준 방식으로, 각국 안전 규격(ATEX, IECEx, KGS 등)에 광범위하게 적용됩니다.
- **우수한 선형성:** LEL 0~100% 전 범위에서 농도-출력 선형성이 높아 정량 측정에 적합합니다.
- **빠른 응답:** 확산 방식 기준 T₉₀ ≤ 30초, 흡인 방식에서는 더욱 단축 가능합니다.
- **가스 비선택적:** 대부분의 가연성 가스에 반응하여 혼합 가스 환경에서도 누출 감지가 가능합니다.

### 접촉연소식 단점 및 설계 주의사항

- **높은 소비 전력:** 비드를 약 500°C로 유지하는 히터 전력이 상시 필요합니다. 배터리 구동 기기에는 불리합니다.
- **촉매 피독(Poisoning):** 실리콘 화합물(Si), 황화물(H₂S, SO₂), 납(Pb) 등에 장기 노출 시 촉매 활성이 저하됩니다.
- **산소 의존성:** 연소 반응에 산소가 필요하므로 산소 결핍 환경(O₂ < 10%vol)에서는 측정값이 부정확해집니다.
- **100% LEL 초과 불감지:** 가스 농도가 LEL을 크게 초과하면 산소 부족으로 연소 반응이 억제되어 출력이 오히려 낮아지는 역전 현상이 발생할 수 있습니다.
- **고농도 가스 노출 시 손상:** 100% LEL 이상의 고농도 가스에 장기 노출되면 비드 과열로 영구 손상이 발생할 수 있습니다.

## 설계 체크리스트

접촉연소식 센서를 회로·시스템에 적용할 때 검토해야 할 주요 항목입니다.

- [ ] **히터 전원 공급:** 비드 가열용 정전압/정전류 공급 회로 설계. 전원 변동이 출력에 직접 영향을 미칩니다.
- [ ] **브리지 구동 전압:** 제조사 권장 구동 전압 준수 (보통 1.5 ~ 2.5 V, 전류 200 ~ 400 mA 수준).
- [ ] **차동 증폭 회로:** 브리지 출력은 수 mV 수준이므로 고정밀 계측 증폭기(INA 시리즈 등) 사용 권장.
- [ ] **워밍업 시간:** 센서 전원 인가 후 측정값이 안정화되기까지 30초 ~ 수 분의 워밍업 시간이 필요합니다.
- [ ] **산소 농도 모니터링:** 저산소 환경 적용 시 O₂ 센서 병행 탑재하여 측정 유효성을 확인합니다.
- [ ] **피독 환경 사전 검토:** 설치 환경의 Si, S 화합물 존재 여부 확인. 필요 시 필터 또는 보호 캡 적용.
- [ ] **방폭 인증 요건:** 설치 환경의 방폭 등급(Zone 0/1/2) 확인 및 해당 인증 취득 제품 선정.
- [ ] **교정 주기:** 6개월 ~ 1년 주기의 정기 교정 계획을 시스템 운영 절차에 포함.
- [ ] **경보 설정 기준:** 1차 경보 10~25% LEL, 2차 경보 40~50% LEL이 일반적. 관련 법규·표준 확인 필수.
