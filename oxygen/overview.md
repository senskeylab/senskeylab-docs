# 산소 센서 개요

산소 센서(Oxygen Sensor)는 공기 또는 특정 환경 내 O₂ 농도를 측정하는 센서입니다. 산소 결핍 감지, 연소 제어, 의료 기기, 식품 포장 등 광범위한 분야에 적용됩니다. 측정 원리에 따라 **전기화학식(갈바닉/폴라로그래픽)**, **지르코니아식**, **광자기식(파라마그네틱)** 등으로 구분됩니다.

---

## 측정 방식 종류

| 방식 | 원리 | 주요 적용 |
|------|------|-----------|
| **전기화학식 (갈바닉)** | O₂ 환원 반응 전류 측정 | 산소 결핍 경보, 휴대형 기기 |
| **전기화학식 (폴라로그래픽)** | 외부 바이어스 인가, O₂ 환원 전류 측정 | 의료용, 정밀 분석 |
| **지르코니아식** | 고온 이온 전도 기전력 측정 | 연소 제어, 배기가스 분석 |
| **광자기식 (파라마그네틱)** | O₂의 강한 상자성 특성 이용 | 고정밀 분석 계기 |

이 페이지에서는 산업 현장과 휴대형 기기에 가장 많이 사용되는 **전기화학식(갈바닉)**을 중심으로 설명하며, 지르코니아식도 함께 다룹니다.

---

## 동작 원리

### 전기화학식 — 갈바닉 방식 (Galvanic)

외부 전원 없이 산화–환원 반응의 자발적 기전력으로 전류를 발생시킵니다.

| 전극 | 반응식 |
|------|--------|
| 캐소드 (WE, 환원) | O₂ + 2H₂O + 4e⁻ → 4OH⁻ |
| 애노드 (CE, 산화) | Pb + 2OH⁻ → PbO + H₂O + 2e⁻  *(납 애노드의 경우)* |

- 외부 전원 불필요 (자발적 반응)
- O₂ 농도에 비례하는 단락 전류 출력
- 수명: 반응 소모 물질(납 애노드 등) 고갈 시 교체 필요 (통상 2~3년)

### 전기화학식 — 폴라로그래픽 방식 (Polarographic)

외부 바이어스 전압(약 −0.6~−0.8 V)을 인가하여 O₂ 환원 반응을 강제 유도합니다.

- 안정성·재현성 우수
- 의료용 혈중 산소 분압(PO₂) 측정, 고정밀 산업용 분석기에 사용

### 지르코니아식 (Zirconia)

ZrO₂ 세라믹을 약 600~850°C로 가열하면 산소 이온 전도체가 됩니다. 측정면과 기준면(대기) 사이의 O₂ 농도 차이에 의해 네른스트 기전력(Nernst EMF)이 발생합니다.

```
E = (RT / 4F) × ln(P_ref / P_sample)
```

- R: 기체 상수, T: 절대 온도, F: 패러데이 상수
- 고온 환경(연소로, 배기가스)에서 실시간 O₂ 측정에 적합
- 응답 속도 매우 빠름 (< 1 s)
- 히터 전력 소모 큼 (수 W)

## 센서 셀 구조 (전기화학식 갈바닉)

```
  ┌─────────────────────────────────────────┐
  │            가스 유입 (O₂ 포함 공기)       │
  └──────────────┬──────────────────────────┘
                 │
  ┌──────────────▼──────────────────────────┐
  │        소수성 가스 확산막                 │  ← O₂ 투과 속도 제어 (선형성 확보)
  └──────────────┬──────────────────────────┘
                 │
  ┌──────────────▼──────────────────────────┐
  │       캐소드 (귀금속, Au/Pt 등)           │  ← O₂ 환원 반응 (전류 발생)
  ├─────────────────────────────────────────┤
  │         전해질 (KOH 수용액 등)            │  ← OH⁻ 이온 이동 매개
  ├─────────────────────────────────────────┤
  │       애노드 (Pb 또는 귀금속)             │  ← 산화 반응 (소모 전극)
  └─────────────────────────────────────────┘
                 │
  ┌──────────────▼──────────────────────────┐
  │       전류-전압 변환 (TIA) 회로           │  ← 단락 전류 → 전압 변환
  └─────────────────────────────────────────┘
```

| 구성 요소 | 역할 |
|-----------|------|
| **캐소드 (WE)** | Au, Pt 등 귀금속. O₂ 환원 반응 발생. 소모되지 않음 |
| **애노드 (CE)** | Pb, Ag 등. 산화 소모됨 → 수명 결정 요소 |
| **전해질** | KOH 수용액 또는 고체 전해질. OH⁻ 이온 이동 경로 |
| **가스 확산막** | PTFE 소수성 막. O₂ 투과 속도를 제한하여 선형 응답 보장 |

## 주요 특성

| 항목 | 전기화학식 (갈바닉) | 지르코니아식 |
|------|------------------|------------|
| **측정 범위** | 0 ~ 30 %vol | 0.01 ~ 21 %vol (또는 ppm 급) |
| **분해능** | 0.1 %vol | 0.01 %vol |
| **T₉₀ 응답시간** | ≤ 15 s | < 1 s |
| **동작 온도** | −20 ~ +50°C | 600 ~ 850°C (세라믹 자체) |
| **소비 전력** | 매우 낮음 (μW ~ mW) | 높음 (히터 수 W) |
| **수명** | 2 ~ 3년 | 5년 이상 |
| **교정 필요** | 6개월 ~ 1년 주기 | 주기적 교정 |
| **주요 적용** | 산소 결핍 경보, 휴대형 | 연소 제어, 배기가스 |

---

## 교차 감응 및 간섭 인자

전기화학식 산소 센서는 아래 환경에서 측정 오차가 발생할 수 있습니다.

| 간섭 인자 | 영향 | 대응 방법 |
|-----------|------|-----------|
| 이산화탄소 (CO₂) 고농도 | 전해질 반응, 감도 변화 | CO₂ 내성 전해질 사용 또는 사전 필터 |
| 할로겐 가스 (Cl₂, F₂) | 캐소드 피독 | 활성탄 필터 적용 |
| 온도 변화 | 반응 속도 변화 → 출력 드리프트 | 온도 보상 회로 또는 NTC 보정 |
| 기압 변화 | O₂ 분압 변화 → 측정값 오차 | 고도 보정 알고리즘 적용 |
| 진동·충격 | 전해질 누출 위험 | 방진 구조 또는 고체 전해질 제품 선택 |

---

## 적용 분야

| 분야 | 용도 |
|------|------|
| **산업 안전** | 밀폐 공간 작업 시 산소 결핍 (O₂ < 18%vol) 경보 |
| **의료** | 인큐베이터, 마취기, 인공호흡기 내 O₂ 농도 제어 |
| **식품 포장** | MAP(변조 대기 포장) 공정 내 O₂ 잔류 검사 |
| **연소 제어** | 보일러·소각로 배기가스 O₂ 측정으로 연소 효율 최적화 (지르코니아식) |
| **반도체 공정** | 불활성 가스 분위기 관리 (ppm 수준 O₂ 모니터링) |
| **수처리** | 용존 산소(DO) 측정 (전기화학식 응용) |

## 설계 체크리스트

- [ ] **측정 목적 확인:** 산소 결핍 경보(18%vol 이하)인지, 정밀 정량(연소 제어)인지에 따라 방식 선택.
- [ ] **TIA 회로:** 갈바닉 방식의 단락 전류(수 μA ~ 수백 μA)를 전압으로 변환하는 고입력 임피던스 TIA 설계.
- [ ] **온도 보상:** 센서 출력은 온도에 따라 변화하므로 NTC 또는 PT100 병행 탑재 후 보정.
- [ ] **기압 보정:** 고도 변화가 있는 환경(드론, 산악 지역)에서는 기압 센서 병행 탑재.
- [ ] **보관 조건:** 갈바닉 방식은 공기와 접촉하는 순간부터 소모가 시작됩니다. 보관 시 밀봉 처리하여 수명 손실 최소화.
- [ ] **수명 잔여량 모니터링:** 애노드 소모를 간접 추정하는 알고리즘(누적 전하량 추적) 구현 권장.
- [ ] **경보 임계값:** 산업안전보건법 기준 O₂ 결핍 경보 18%vol, 위험 경보 16%vol 이하 확인.
- [ ] **지르코니아식 적용 시:** 히터 제어 회로, 고온 배선(내열 케이블), 기준 공기 공급 구조 별도 설계 필요.
