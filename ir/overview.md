# 적외선식 가스 센서 개요 (NDIR)

적외선식 가스 센서(Non-Dispersive Infrared Sensor, NDIR)는 가스 분자가 특정 파장의 적외선을 흡수하는 성질을 이용하여 농도를 측정합니다. 촉매 피독이 없고 수명이 길며 선택성이 우수하여 CO₂, CH₄, CO, 탄화수소류 측정에 널리 사용됩니다.

---

## 동작 원리

가스 분자는 고유한 분자 구조에 따라 특정 적외선 파장을 흡수합니다(분자 진동 공명). 이 흡수량은 **람베르트-비어 법칙(Beer-Lambert Law)**에 따라 가스 농도에 비례합니다.

**람베르트-비어 법칙**

```
I = I₀ × e^(−α × c × L)
```

| 기호 | 의미 |
|------|------|
| I | 투과 후 광 강도 |
| I₀ | 초기 광 강도 |
| α | 흡광 계수 (가스 고유값) |
| c | 가스 농도 |
| L | 광경로 길이 |

농도가 높을수록 적외선 흡수량이 많아져 검출기에 도달하는 광 강도가 줄어듭니다. 이 감쇠량으로 가스 농도를 계산합니다.

**주요 가스별 IR 흡수 파장**

| 가스 | 주 흡수 파장 | 특징 |
|------|------------|------|
| CO₂ | 4.26 μm | 가장 강한 흡수 밴드, 측정 용이 |
| CO | 4.67 μm | CO₂와 근접하여 광학 필터 설계 중요 |
| CH₄ | 3.31 μm | 탄화수소 C-H 결합 흡수 |
| C₃H₈ | 3.37 μm | CH₄과 근접, 다중 가스 측정 시 주의 |
| N₂O | 4.47 μm | 의료·환경 모니터링 |
| SO₂ | 7.35 μm | 환경 배출 모니터링 |

> **비 IR 흡수 가스:** H₂, O₂, N₂, Cl₂ 등 단원자 또는 대칭 이원자 분자는 IR 흡수가 없어 NDIR로 측정 불가능합니다.

## 센서 구조

```
  ┌─────────────────────────────────────────────────────────┐
  │                                                         │
  │  [IR 광원]──────[광경로(가스 셀)]──────[광학 필터]──[검출기] │
  │   (흑체 또는     (반사경 구조로       (특정 파장만     (열전쌍,  │
  │    LED)         광경로 연장)          투과)         초전형 등) │
  │                                                         │
  │        ┌──────────────────────────┐                     │
  │        │     기준 채널 (Reference) │  ← 가스 비흡수 파장   │
  │        │     측정 채널 (Active)    │  ← 가스 흡수 파장     │
  │        └──────────────────────────┘                     │
  └─────────────────────────────────────────────────────────┘
                         │
           ┌─────────────▼──────────────┐
           │  신호 처리 (기준/측정 비율)   │  ← 광원 드리프트 보상
           └─────────────────────────────┘
```

| 구성 요소 | 역할 |
|-----------|------|
| **IR 광원** | 흑체 복사 광원(백열 필라멘트) 또는 MEMS IR LED. 넓은 파장 방출 |
| **가스 셀** | 측정 가스가 통과하는 광경로. 반사경으로 광경로를 연장하여 감도 향상 |
| **광학 필터** | 측정 대상 가스의 흡수 파장만 통과시키는 협대역 필터 |
| **검출기** | 투과 IR 강도 측정. 열전쌍(thermopile), 초전형(pyroelectric) 등 |
| **기준 채널** | 가스가 흡수하지 않는 파장을 기준으로 광원 드리프트·오염을 보상 |

### 듀얼 채널 구조의 중요성

단일 채널 구성은 광원 노화, 렌즈 오염, 온도 변화에 의한 출력 드리프트에 취약합니다. **기준 채널(Reference Channel)**을 병행하여 두 채널의 비율로 농도를 계산하면 이러한 드리프트를 효과적으로 제거할 수 있습니다.

## 주요 가스별 특성

| 측정 가스 | 측정 범위 | 분해능 | T₉₀ 응답시간 | 기대 수명 |
|-----------|----------|--------|------------|----------|
| CO₂ | 0 ~ 5,000 ppm / 0 ~ 100 %vol | 1 ppm | ≤ 60 s | 5년 이상 |
| CO | 0 ~ 1,000 ppm | 1 ppm | ≤ 60 s | 5년 이상 |
| CH₄ | 0 ~ 100% LEL / 0 ~ 100 %vol | 0.1% LEL | ≤ 30 s | 5년 이상 |
| C₃H₈ | 0 ~ 100% LEL | 0.1% LEL | ≤ 30 s | 5년 이상 |
| N₂O | 0 ~ 1,000 ppm | 1 ppm | ≤ 60 s | 5년 이상 |

## 장단점 및 타 방식과 비교

| 항목 | NDIR | 접촉연소식 | 반도체식 | 전기화학식 |
|------|------|-----------|---------|-----------|
| **선택성** | 매우 높음 (파장 고유성) | 낮음 | 낮음 | 높음 |
| **촉매 피독** | 없음 | 있음 | 없음 | 일부 |
| **수명** | 5년 이상 | 3~5년 | 3~5년 | 1~3년 |
| **소비 전력** | 보통 (광원 히터) | 높음 | 보통 | 매우 낮음 |
| **측정 범위** | ppm ~ %vol | 0~100% LEL | ppm~% | ppm~%vol |
| **저농도 감지** | 보통 (ppm) | 어려움 | 우수 (ppb~ppm) | 우수 |
| **H₂ 측정** | 불가 | 가능 | 가능 | 불가 |
| **가격** | 높음 | 보통 | 낮음 | 보통~높음 |
| **교정 주기** | 1~2년 | 6개월~1년 | 6개월~1년 | 6개월~1년 |
| **온습도 영향** | 낮음 | 낮음 | 높음 | 보통 |

### NDIR 장점

- **피독 없음:** 화학 반응을 이용하지 않으므로 촉매 피독, 전극 열화가 없습니다.
- **긴 수명:** 광학 부품의 물리적 노화만 고려하면 되므로 5년 이상 운용이 일반적입니다.
- **높은 선택성:** 파장 고유성으로 간섭 가스 영향이 적어 복잡한 환경에서도 신뢰성 높은 측정이 가능합니다.
- **넓은 측정 범위:** ppm 수준부터 %vol 수준까지 동일 원리로 측정 가능합니다.

### NDIR 단점 및 설계 주의사항

- **단원자·대칭 이원자 가스 측정 불가:** H₂, O₂, N₂, Cl₂ 등은 측정 불가능합니다.
- **수분(H₂O) 간섭:** 수증기도 넓은 IR 흡수 밴드를 가집니다. 고습 환경에서는 수분 보정이 필요하며 드라이어 또는 투과막 적용을 검토합니다.
- **가격:** 광원, 광학 필터, 검출기 등 광학 부품 비용으로 인해 타 방식 대비 고가입니다.
- **광경로 오염:** 셀 내부 먼지·오일 미스트가 광량을 감소시켜 측정 오차를 유발합니다. 주기적 청소 또는 방진 필터 적용이 필요합니다.
- **응답 속도:** 가스 셀 내 가스 치환 시간 때문에 확산 방식에서는 응답이 다소 느립니다. 흡인 펌프 적용으로 단축 가능합니다.

## 설계 체크리스트

- [ ] **광원 선택:** 백열 필라멘트(광량 크나 수명 ~5년, 소비 전력 수백 mW) vs MEMS IR LED(저전력, 고속 변조 가능, 소형).
- [ ] **광원 변조(Modulation):** 광원을 1~10 Hz로 변조하면 검출기 DC 노이즈를 제거하고 S/N비를 향상시킬 수 있습니다.
- [ ] **듀얼 채널 구성:** 기준 채널 병행으로 광원 드리프트·오염 보상. 단일 채널 구성은 장기 안정성에 불리합니다.
- [ ] **광학 필터 선정:** 측정 가스 흡수 파장에 맞는 협대역 필터(FWHM 100~200 nm 수준) 선정. CO와 CO₂처럼 근접 파장 가스는 필터 특성에 주의.
- [ ] **가스 셀 설계:** 광경로 길이(L)가 길수록 감도 향상. 다중 반사 구조(White cell, Herriott cell)로 소형화 가능.
- [ ] **수분 대응:** 고습 환경이면 가스 셀 전단에 투과 필터(PTFE 등) 또는 드라이어 튜브 적용.
- [ ] **온도 제어:** 검출기(특히 초전형) 감도가 온도에 민감하므로 온도 보상 또는 검출기 온도 안정화 회로 검토.
- [ ] **교정:** CO₂ 등 대기 중 농도가 있는 가스는 기준 농도 교정이 중요합니다. 자동 영점(Auto-zero) 기능 구현 권장.
