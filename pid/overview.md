# PID 센서 개요 (광이온화 검출기)

PID(Photoionization Detector, 광이온화 검출기)는 자외선(UV) 광자로 가스 분자를 이온화하여 발생하는 미세 전류로 가스 농도를 측정합니다. ppb ~ ppm 수준의 극저농도 VOC(휘발성 유기화합물) 검출에 특화되어 있으며, 응급 대응, 산업위생, 환경 모니터링 분야에서 필수 장비입니다.

---

## 동작 원리

UV 광원이 방출하는 광자(photon)가 가스 분자에 충돌하면, 분자의 이온화 전위(Ionization Potential, IP)보다 큰 에너지를 가진 광자에 의해 분자가 이온화됩니다.

```
M + hν → M⁺ + e⁻
```

| 기호 | 의미 |
|------|------|
| M | 가스 분자 |
| hν | UV 광자 에너지 (h: 플랑크 상수, ν: 주파수) |
| M⁺ | 양이온 |
| e⁻ | 자유 전자 |

생성된 이온 쌍이 전극 사이의 전기장에 의해 이동하며, 이때 흐르는 전류가 가스 농도에 비례합니다.

**UV 램프 에너지와 감지 가능 가스**

| UV 램프 에너지 | 주요 감지 가능 가스 |
|--------------|------------------|
| **10.6 eV** | 대부분의 VOC (벤젠, 톨루엔, 자일렌, 에틸렌 등). 가장 범용적 |
| **10.0 eV** | 방향족 탄화수소 선택 감지. 암모니아, 황화수소도 감지 |
| **11.7 eV** | 염소화 용제, 에탄, 메탄 등 포함. 최고 에너지 (창 재질: LiF) |

> **핵심:** 가스 분자의 이온화 전위(IP)가 UV 램프 에너지보다 낮아야 감지 가능합니다. IP가 높은 메탄(12.6 eV), 에탄(11.5 eV)은 10.6 eV 램프로 감지되지 않습니다.

## 센서 구조

```
  ┌─────────────────────────────────────────────────────┐
  │                 가스 유입 (확산 또는 흡인)             │
  └──────────────────────┬──────────────────────────────┘
                         │
  ┌──────────────────────▼──────────────────────────────┐
  │                                                     │
  │   [UV 램프]                                          │
  │  (10.6/10.0/11.7 eV)                                │
  │       │  UV 광자 방출                                │
  │       ▼                                              │
  │  ┌─────────────────┐                                │
  │  │   이온화 챔버     │  ← 가스 분자 이온화 발생       │
  │  │  [+극] [−극]    │  ← 전기장 형성 (이온 수집)     │
  │  └────────┬────────┘                                │
  │           │ 이온 전류 (pA ~ nA)                      │
  └───────────┼─────────────────────────────────────────┘
              │
  ┌───────────▼─────────────────────────────────────────┐
  │      전류 증폭 및 신호 처리 (전류계, ADC)              │
  └─────────────────────────────────────────────────────┘
```

| 구성 요소 | 역할 |
|-----------|------|
| **UV 램프** | 크립톤(Kr) 또는 아르곤(Ar) 기체 봉입 방전 램프. 에너지 레벨별 라인업 존재 |
| **UV 투과 창 (Window)** | MgF₂(11.7 eV), LiF(11.7 eV), CaF₂(10.0 eV) 등. 에너지 결정 요소 |
| **이온화 챔버** | 가스 분자가 UV에 노출되는 공간. 수집 전극 내장 |
| **수집 전극** | 이온화된 양이온과 전자를 각각 수집하여 전류 형성 |
| **증폭 회로** | pA ~ nA 수준의 극미세 전류를 처리 가능한 수준으로 증폭 |

## 주요 특성

| 항목 | 사양 |
|------|------|
| **측정 범위** | 0 ~ 10 ppm (고감도) / 0 ~ 1,000 ppm (광범위) |
| **검출 하한 (LOD)** | 1 ppb 이하 (가스 종류에 따라 다름) |
| **분해능** | 0.1 ppb ~ 1 ppb |
| **T₉₀ 응답시간** | ≤ 3 s (흡인식), ≤ 10 s (확산식) |
| **동작 온도** | −20 ~ +50°C |
| **UV 램프 수명** | 5,000 ~ 10,000시간 |
| **소비 전력** | 낮음 ~ 보통 (UV 램프 종류에 따라 수십~수백 mW) |

## 감도 보정 계수 (Response Factor)

PID 센서는 보통 **이소부틸렌(isobutylene)**을 기준으로 교정(교정 계수 = 1.0)하며, 다른 가스의 농도는 상대 감도(Response Factor)를 적용하여 환산합니다.

```
실제 농도 = 측정값(ppm) × Response Factor
```

| 가스 | Response Factor (대 이소부틸렌) |
|------|-------------------------------|
| 이소부틸렌 | 1.0 (기준) |
| 벤젠 | 0.53 |
| 톨루엔 | 0.54 |
| 자일렌 | 0.43 |
| 에틸렌 | 1.07 |
| 암모니아 | 9.7 |
| H₂S | 3.7 |
| 아세톤 | 1.1 |

> **주의:** Response Factor는 제조사와 UV 램프 에너지에 따라 다릅니다. 정확한 측정을 위해서는 측정 대상 가스로 직접 교정하거나 제조사 제공 계수를 사용하십시오.

## 간섭 인자

| 간섭 인자 | 영향 | 대응 방법 |
|-----------|------|-----------|
| **고습도** | 수분이 UV 흡수 → 감도 저하 | 내습성 설계, 습도 보정 알고리즘, 흡인+드라이어 |
| **산소 농도** | O₂가 UV 흡수 → 측정값 변화 | 산소 보정 알고리즘 또는 일정 O₂ 농도 환경 유지 |
| **UV 창 오염** | 감도 저하, 영점 드리프트 | 주기적 창 청소 또는 교체 |
| **혼합 VOC 환경** | 개별 성분 구분 불가 | GC-PID(가스 크로마토그래피 연동) 또는 VOC 총량 측정으로 용도 한정 |
| **온도** | UV 램프 방출 특성 변화 | 온도 보상 회로 |

## 적용 분야

| 분야 | 용도 |
|------|------|
| **산업위생** | 작업 환경 내 VOC 농도 모니터링 (TLV-TWA 기준 준수) |
| **응급 대응 (HazMat)** | 화학 사고 현장에서 미지 유해 물질 신속 탐지 |
| **환경 모니터링** | 토양 오염 조사, 매립지 가스 모니터링 |
| **산업 공정** | 페인트, 도료, 세정 공정 VOC 배출 제어 |
| **누설 감지** | 배관·탱크 접합부 VOC 누설 위치 탐지 |
| **실내 공기질 (IAQ)** | TVOC 모니터링 (건축 자재·가구 방출 VOC) |

## 타 방식과 비교

| 항목 | PID | 전기화학식 | 반도체식 | NDIR |
|------|-----|-----------|---------|------|
| **주요 대상** | VOC (광범위) | 특정 독성 가스 | 가연성·독성 가스 | CO₂, CH₄ 등 |
| **감지 하한** | ppb 수준 | ppm 수준 | ppm ~ ppb | ppm 수준 |
| **선택성** | 낮음 (VOC 전반 감지) | 높음 | 낮음 | 매우 높음 |
| **CH₄, H₂ 감지** | 불가 (10.6 eV) | 불가 | 가능 | 가능 (NDIR) |
| **수명** | UV 램프 교체 필요 | 1~3년 | 3~5년 | 5년 이상 |
| **소비 전력** | 낮음~보통 | 매우 낮음 | 보통 | 보통 |
| **가격** | 높음 | 보통~높음 | 낮음 | 높음 |
| **습도 영향** | 높음 | 보통 | 높음 | 낮음 |

## 설계 체크리스트

- [ ] **UV 램프 에너지 선정:** 측정 대상 VOC의 이온화 전위(IP) 목록을 확인하고, 모든 대상의 IP를 커버하는 램프 에너지 선택.
- [ ] **전류 측정 회로:** 이온 전류는 pA ~ nA 수준이므로 전용 피코암페어 측정 IC 또는 고입력 임피던스 TIA 필요. PCB 누설 전류 최소화 설계 필수.
- [ ] **UV 창 보호:** 창 오염이 감도에 직접 영향. 측정 환경의 분진·오일 미스트 여부 확인 후 필터 또는 흡인 방식 적용.
- [ ] **습도 대응:** 고습 환경에서 감도 저하가 큼. 흡인 방식 + 드라이어 또는 습도 보정 알고리즘 구현.
- [ ] **교정 가스:** 이소부틸렌 100 ppm 표준 가스로 정기 교정. 측정 대상 가스의 Response Factor 데이터시트 확인.
- [ ] **UV 램프 수명 관리:** 누적 점등 시간을 펌웨어에서 추적하고 5,000~10,000시간 도달 시 교체 알림 구현.
- [ ] **산소 농도 보정:** 산소 결핍 또는 고산소 환경에서 측정값 보정이 필요한지 검토.
- [ ] **혼합 가스 환경:** PID는 VOC 총량(TVOC)을 반환하며 개별 성분을 구분하지 못합니다. 개별 성분 분석이 필요하면 GC 연동을 검토.
