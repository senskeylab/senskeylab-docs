# 촉매 연소식 vs 반도체식 가스 센서 비교

가연성 가스 감지에 가장 널리 사용되는 두 방식인 **접촉연소식(Catalytic Bead)**과 **반도체식(Metal Oxide Semiconductor, MOS)**은 동작 원리와 특성이 근본적으로 다릅니다. 이 문서는 두 방식의 원리부터 설계 적용 판단 기준까지 엔지니어링 관점에서 정리합니다.

---

## 동작 원리 비교

### 접촉연소식 (Catalytic Bead)

가연성 가스가 Pt/Pd 촉매 코팅된 비드 표면에서 저온 산화 연소를 일으키고, 발생한 연소열이 비드 내 코일의 전기 저항을 변화시킵니다. 이 저항 변화를 휘트스톤 브리지로 검출하여 농도를 측정합니다.

```
가스 + O₂ ──[촉매, ~500°C]──→ CO₂ + H₂O + 열
                                          │
                                  Detector bead 저항 증가
                                          │
                                  브리지 불평형 전압 출력
```

- **출력 유형:** 전압 (mV, 브리지 차동 출력)
- **측정 단위:** % LEL (0 ~ 100% LEL)
- **산소 의존성:** 있음 (연소 반응에 O₂ 필수)

### 반도체식 (MOS)

SnO₂, ZnO, In₂O₃ 등 금속 산화물 반도체를 약 200~400°C로 가열하면 표면의 흡착 산소(O⁻, O²⁻)가 가연성 가스와 반응하여 전도 전자를 방출하고, 센서의 전기 저항이 감소합니다.

```
가스 (환원성) + O⁻(표면) → 산화 생성물 + e⁻
                                         │
                               반도체 저항 감소
                                         │
                               저항비(R_air / R_gas) 출력
```

- **출력 유형:** 저항 변화 (또는 전압 분배 출력)
- **측정 단위:** ppm 또는 %vol (비선형 특성 있음)
- **산소 의존성:** 간접적 영향 (표면 흡착 산소량에 따라 감도 변화)

---

## 주요 특성 비교

| 항목 | 접촉연소식 | 반도체식 (MOS) |
|------|-----------|--------------|
| **측정 범위** | 0 ~ 100% LEL | ppm ~ %(vol) 광범위 |
| **선형성** | 우수 (전 범위 선형) | 낮음 (로그 특성, 농도 보정 필요) |
| **선택성** | 낮음 (가연성 가스 전반 반응) | 매우 낮음 (습도·온도·다양한 가스 간섭) |
| **감도 (저농도)** | 낮음 (수십 ppm 이하 검출 어려움) | 높음 (수 ppm ~ 수십 ppb 검출 가능) |
| **소비 전력** | 높음 (500 mW ~ 1 W) | 보통 (100 ~ 300 mW, 히터 포함) |
| **응답 속도** | 빠름 (T₉₀ ≤ 30 s) | 빠름 (T₉₀ 10 ~ 30 s) |
| **회복 속도** | 빠름 | 보통 ~ 느림 (고농도 노출 후 드리프트) |
| **출력 안정성** | 높음 | 낮음 (장기 드리프트, 초기 번인 필요) |
| **촉매 피독** | 있음 (Si, S, Pb, Cl) | 없음 |
| **고농도 손상** | 있음 (100% LEL 초과 시 비드 과열) | 내성 있음 |
| **산소 의존성** | 높음 (O₂ < 10% 시 측정 불가) | 낮음 |
| **수명** | 3 ~ 5년 | 3 ~ 5년 (환경에 따라 편차 큼) |
| **온도 영향** | 낮음 (Compensator로 보상) | 높음 (감도가 온도에 민감) |
| **습도 영향** | 낮음 | 높음 (수분이 표면 흡착 산소에 영향) |
| **가격** | 보통 | 낮음 |
| **크기** | 보통 | 소형화 가능 |
| **방폭 구조** | 소결 금속 필터 필수 | 케이스 설계로 대응 |

## 감도 및 선형성 특성

### 접촉연소식 — 선형 출력

```
출력(mV)
   ^
   |                              /
   |                           /
   |                        /
   |                     /
   |                  /
   |               /
   |            /
   |_________/______________________________→ 농도 (% LEL)
   0         25        50        75       100
```

농도–출력이 거의 선형이어서 교정 및 농도 환산이 단순합니다.

### 반도체식 — 로그 특성 출력

```
저항비 (Rs/R0)
   ^
   |
10 |\ 
   | \
 5 |   \
   |     \
 2 |       \___
   |            \____
 1 |                  \________
   |_____________________________→ 농도 (ppm)
   1    10   100  1000  10000
```

저항 변화가 농도의 로그에 비례합니다. 저농도 변화는 크게 나타나지만 고농도에서 포화됩니다. 정밀 정량에는 교정 커브 또는 다항식 보정이 필요합니다.

## 환경 영향 인자

### 온도 영향

| 방식 | 영향 | 대응 방법 |
|------|------|-----------|
| 접촉연소식 | 작음 (Compensator bead가 환경 온도 변화 보상) | 광범위 온도 환경에서도 안정적 |
| 반도체식 | 큼 (히터 온도와 감지층 상태가 온도에 민감) | 온도 보정 알고리즘, 온도 제어 히터 |

### 습도 영향

| 방식 | 영향 | 대응 방법 |
|------|------|-----------|
| 접촉연소식 | 작음 | 일반적으로 별도 보정 불필요 |
| 반도체식 | 큼 (수분이 표면 흡착 산소 밀도를 변화시킴) | 습도 센서 병행 탑재 후 보정 |

### 촉매 피독 (접촉연소식 전용 이슈)

| 피독 물질 | 주요 발생 환경 | 영향 |
|-----------|-------------|------|
| 실리콘 화합물 (Si) | 실리콘 씰런트, 코팅제, 플라스틱 | 촉매 비활성화 → 감도 영구 저하 |
| 황화물 (H₂S, SO₂) | 황 함유 가스 환경 | 촉매 피독 → 감도 저하 |
| 납 (Pb) | 배터리 환경, 납 함유 연료 | 촉매 비활성화 |
| 염소 화합물 (Cl) | 세척제, 냉매 | 촉매 피독 |

> **실무 팁:** 실리콘 씰런트를 센서 근처에서 사용하면 경화 과정에서 발생하는 실록세인(siloxane) 증기가 촉매를 피독시킵니다. 설치 환경에서 실리콘 계열 재료 사용 여부를 반드시 확인하십시오.

## 적용 시나리오별 권장 방식

| 적용 시나리오 | 권장 방식 | 이유 |
|-------------|---------|------|
| 산업 현장 고정식 가스 검지기 (LEL 기준) | **접촉연소식** | 선형성·신뢰성 우수, 법적 인증 적합 |
| 가정용 가스 경보기 (메탄, LPG) | **반도체식** | 저가, 소형, ppm 수준 누출 조기 감지 |
| 배터리 구동 휴대형 검지기 | **반도체식** | 저전력, 소형화 유리 |
| 저농도(ppm) 가스 조기 감지 | **반도체식** | 고감도, 저농도 반응성 우수 |
| 고농도(% LEL) 정량 측정 | **접촉연소식** | 전 범위 선형, 정확도 우수 |
| 산소 결핍 가능 환경 | **반도체식** | 산소 의존성 없음 |
| 실리콘·황 화합물 노출 환경 | **반도체식** | 촉매 피독 없음 |
| 방폭 인증(ATEX/IECEx/KGS) 필요 | **접촉연소식** | 오랜 인증 실적, 규격 적합 제품 다양 |
| 다종 가스 동시 감지 (e-nose) | **반도체식 어레이** | 복수 센서 패턴 인식으로 가스 판별 가능 |

## 요약

| | 접촉연소식 | 반도체식 |
|--|-----------|---------|
| **선택 기준** | 산업용 정량 측정, LEL 기준, 방폭 인증 필요 | 저가·소형·저전력, 저농도 조기 감지 |
| **피해야 할 환경** | 산소 결핍, 실리콘·황 화합물 존재 | 고온·고습 변동 환경, 정밀 정량 필요 |
| **회로 복잡도** | 높음 (정밀 전원, 계측 증폭기 필요) | 낮음 (저항 분배 또는 간단한 ADC) |
| **유지보수** | 정기 교정 필수 (6개월~1년) | 교정 주기 길거나 자동 보정 가능 |
